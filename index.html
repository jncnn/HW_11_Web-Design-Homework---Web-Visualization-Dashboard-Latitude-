SUMMARY:LATITUDE VS X

The purpose of the this project was to analyze how weather changes as you change latitude. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to create data set on over 500 cities as csv file.(cities.csv) We used cities.csv file to and matplotlib and plot latitude verses temperature, cloudiness,windspeed, and humidity.
